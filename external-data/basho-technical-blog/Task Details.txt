Convert Technical Blog HTML pages to Markdown

Extract:
- Title: og:title
- Description: og:description
- Date: article:published_time
- Author: <p class="byline author vcard"><a>by ABCD</a></p>
- Content: <article><div class="entry-content">...* </article>


Convert:

H1 -> ##
H2 -> ###
H3 -> ####
p -> concatenate and end with CRLF
code -> `...`
pre -> ``` ... ```

ul -> add 2 spaces to indent
li -> - text
/ul -> remove 2 spaces from indent

ol -> add 2 spaces to indent
li -> 1. text
/ol -> remove 2 spaces from indent

a -> [text](href)
