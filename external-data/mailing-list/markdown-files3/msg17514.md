We have released Riak TS 1.3.1 today. This is a bugfix release addressing
the data loss bug [1] in Riak TS 1.3.0. While not all users were impacted
by this issue, we recommend that all 1.3.0 users upgrade to 1.3.1. Release
Notes [2] and downloads [3] are now available

[1] http://docs.basho.com/community/productadvisories/130-dataloss/
[2] http://docs.basho.com/riak/ts/1.3.1/releasenotes/
[3] http://docs.basho.com/riak/ts/1.3.1/downloads/
