As far as I know, the lsb\_release tool is installed by default on all
Ubuntu installs. You can use it to get the codename of the release like:

For Ubuntu 10.04

$ lsb\_release -c
Codename: lucid

or Ubuntu 12.04

$ lsb\_release -c
Codename: precise

You can also get the information from the /etc/lsb-release file directly.

% cat /etc/lsb-release
DISTRIB\_ID=Ubuntu
DISTRIB\_RELEASE=12.04
DISTRIB\_CODENAME=precise
DISTRIB\_DESCRIPTION="Ubuntu 12.04.1 LTS"


Hope that helps,
Jared
