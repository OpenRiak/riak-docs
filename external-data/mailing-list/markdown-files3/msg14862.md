> Hi tele,

> Yes, a secondary index is the most reasonable way to accomplish this.
> Here's an example using the Python client:
>
> now = time.gmtime()
> myobj.add\_index('modified\_int', now)
> myobj.store()
>
> bucket.get\_index('modified\_int', now-3600, now+3600)
>
> Hope that helps.
