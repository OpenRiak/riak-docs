Gavin,

Disk space won't be released until Riak's bitcask backend does a merge. You
can tune merge settings by following the instructions here:

http://docs.basho.com/riak/latest/ops/advanced/backends/bitcask/

Also, please see the discussion here:

http://lists.basho.com/pipermail/riak-users\_lists.basho.com/2013-May/012134.html
