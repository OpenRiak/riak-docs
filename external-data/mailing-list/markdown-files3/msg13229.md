Hi,

Riak exposes a `platform\_data\_dir` setting that allows you to tell it
where to store your data. Unfortunately, there is currently no support
for providing multiple data directories (outside of a multi\_backend
scenario).

In practice, users use mdadm and LVM or ZFS to create a single volume
that can be mounted as the Riak data directory.
