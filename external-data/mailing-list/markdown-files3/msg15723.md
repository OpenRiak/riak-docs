Resending the announcement email, but with the correct release this
time in the subject :)

Riak CS 1.5.4 has been released and can be downloaded from the
downloads page[1]. This release contains bug fixes. Please review the
Release Notes [2] for more details before upgrading.

Fixes


 - Disable backpressure sleep riak\_cs/#1041

 - Fix an incorrect path rewrite in the S3 API caused by unnecessary
URL decoding riak\_cs/#1040
