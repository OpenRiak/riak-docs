We are pleased to announce a bugfix release on the Riak 1.4 series addressing 
issues with Riak Control, Secondary Indexes and LevelDB.

The release notes for Riak can be found here: 
https://github.com/basho/riak/blob/1.4/RE LEASE-NOTES.md

The packages can be found here: http://docs.basho.com/riak/1.4.1/downloads/

Users who prefer to build from source, use the riak-1.4.1 tag from 
https://github.com/basho/riak.git
