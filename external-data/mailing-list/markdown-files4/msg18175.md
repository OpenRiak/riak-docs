&gt; @Wagner, Riak KV is a Key Value store with a REST API and client
&gt; implementations in a number of supported client libraries. Riak TS is a
&gt; variant of Riak KV optimised for time series data. Riak TS also has a SQL
&gt; like query language (exposed via riak-shell), which can make some time
&gt; series queries easier to write for end users.
&gt;
&gt; If you want to check out the basic CRUD operations in Riak KV, please
&gt; follow the "Getting Started" section of Riak's documentation [0]. There are
&gt; examples for many programming languages [1][2][3][4][5].
