Based on your previous emails, you have riak-yokozuna installed in
/home/houm/riak-yokozuna. If you use the tech preview, it will be installed
system-wide and won't overwrite your existing installation (which is
probably a good thing).

I would recommend installing the tech preview. The configuration file will
be at /etc/riak/riak.conf (or /usr/local/etc/riak/riak.conf on FreeBSD)
