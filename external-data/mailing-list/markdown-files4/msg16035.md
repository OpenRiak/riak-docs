&gt;&gt; &gt;&gt; examples in our documentation -
&gt;&gt; &gt;&gt; http://docs.basho.com/riak/latest/dev/advanced/mapreduce
&gt;&gt; &gt;&gt;
&gt;&gt; &gt;&gt; I haven't looked at the Python PB API in a while, but I'm reasonably
&gt;&gt; &gt;&gt; certain it supports the invocation of MapReduce jobs.
