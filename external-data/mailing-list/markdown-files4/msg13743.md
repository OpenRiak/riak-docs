&gt;&gt; Hi Abhishek,

&gt;&gt; The Riak Python client does not yet have official support for the
&gt;&gt; version of Riak Search in Riak 2.0.0pre11. That said, you should be
&gt;&gt; able to add values to a Riak 2.0 cluster (with search configured and
&gt;&gt; enabled) and then query those values using a Python Solr client.
&gt;&gt;
&gt;&gt; In addition, to enable search in Riak 2.0.0pre11, you need to edit the
&gt;&gt; `riak.conf` configuration file. Inside `riak.conf` look for `search =
&gt;&gt; off` and switch that to `search = on`. After that, please see the Riak
&gt;&gt; Search simple setup section in our online documentation for further
&gt;&gt; details on indexing and querying data. [0]
&gt;&gt;
&gt;&gt; --
&gt;&gt; Hector
&gt;&gt;
&gt;&gt; [0] http://docs.basho.com/riak/2.0.0pre11/dev/using/search/#Simple-Setup
