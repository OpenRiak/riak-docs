&gt; Hi Eugene,
&gt;
&gt; Which client are you using? Which Riak version?
&gt; In the Java client it is possible to send an update without fetching a
&gt; CRDT first.
&gt;
&gt; For example:
&gt;
&gt; MapUpdate changes = new MapUpdate()
&gt; .update(ARTIFACT, new RegisterUpdate(artifact))
&gt; .update(VERSIONS, new SetUpdate().add(version));
&gt; UpdateMap update = new UpdateMap.Builder(location, changes).build();
&gt; client.execute(update);
&gt;
&gt; Also, take into account that there is a practical limitation of the size
&gt; of a CRDT. Usually it's up to 100-200 KB of raw data.
&gt;
&gt; Regards,
&gt; Vitaly
