Hi Stanislav,

You can change the timeout for a MapReduce job the storage calculation
uses. Could you try to add riakc section which contains mapred\_timeout
into Riak CS's app.config like this ? :

```
[
%% riakc section
 {riakc, [
 {mapred\_timeout, 1800000}], %% msec
 },

[
 %% Riak CS section
 {riak\_cs, [


```

Thanks,
Kaz
