x2, I'd like to know what was the cause of your problem so we could make sure 
it isn't a problem for other people.

In the future, if you want to play with a development cluster on a single 
machine, you can clone from github and do a `make devrel` which will create 
four nodes in their own directories under the `dev` directory.

Sorry for your troubles earlier, but I'm glad you found a solution.
