On Mon, Mar 4, 2013 at 9:36 AM, vvsanil  wrote:

&gt; Is there anyway to set the default search schema analyzer\_factory to
&gt; 'standard\_analyzer\_factory' for all future buckets (i.e. without having to
&gt; manually set schema each time a new bucket is created) ?
&gt;

Yes, look for the file `default.def` under your lib dir where
`riak\_search/priv` lives. That file is used as the default schema when one
is not explicitly created. N.B. you must update this file on \_EVERY\_ node
or you may get unexpected results.
