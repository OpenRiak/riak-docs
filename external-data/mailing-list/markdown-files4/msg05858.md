On 17 Nov 2011, at 13:23, vijayakumar wrote:

&gt; Hi,
&gt; I am using riak java client 0.14 version to insert records, but I am not 
&gt; able to locate any method or options to specify write consistency level. 
&gt; Store method has the following signature store(RiakObject obj). Am I missing 
&gt; anything else? Kindly help me out.

Ok, so in 0.14 the PB and HTTP client don't share an interface but have 
mirrored signatures, really. Both have a second, overloaded store method that 
takes a second parameter: RequestMeta. This RequestMeta lets you set W/DW 
values.

Also, the 1.0.x version of the Riak Java Client has been out for a while now, 
and amongst other things, you get a common API for HTTP and PB. If you use the 
Bucket interface then you set W/DW/PW on the StoreObject builder like this:

bucket.store(myObject).w(3).dw(1).execute();
