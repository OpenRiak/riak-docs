&gt; Hey everyone! As always, there's been some great knowledge sharing over the
&gt; user lists. Here is your biweekly Recap!
&gt;
&gt; Note that many of the "unanswered" are brand new and I know we'll get to
&gt; answering them. I just want you all to know we keep track of these.
&gt;
&gt; ## Announcements
&gt; \* There's a new Java client in town! 2.0.2 was released [0]
&gt;
&gt; ## Recently Answered
&gt; \* Roman asks about using a FQDN and is guided to use 0.0.0.0 instead [1]
&gt; \* The Riak C client ran into a segfault that Chris explained [2]
&gt; \* Humberto found setting ulimit on OS X is a little tricky [3]
&gt; \* I gave Sevilha some resources to spin up Riak locally [4]
&gt; \* Zeeshan makes sense of indexing Riak Data Types for Marius [5]
&gt;
&gt; ## New and Unanswered
&gt; \* Damien needs some help thinking through ring-resizing on a live cluster
&gt; [6]
&gt; \* Sean gets an error while deleting an object through the Erlang client [7]
&gt; \* Humberto could still use help with ulimit settings on OS X [8]
&gt; \* Nick is helping Johan with a known Erlang bug in need of a patch [9]
&gt; \* Damien sees 2i writes when he isn't using 2i. Any idea why? [10]
&gt; \* Amao sees an outbound handoff error and still needs some help [11]
&gt;
&gt; ## All the Meetups!
&gt; We've been busy on the meetup scene, hosting events in Portland, Austin and
&gt; Chicago. If you'd like to see where groups are, we have a new URL to do so
&gt; [12]. Let me know if you'd like to run one in your city!
&gt;
&gt; ## What'd we miss?
&gt; Have you brought Riak into production at a new place? Did you find a blog
&gt; or presentation helpful? Good! Share it in our release notes:
&gt; https://github.com/basho-labs/the-riak-community/tree/master/release-notes
&gt;
