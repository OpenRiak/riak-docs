Henry:

It looks like there is a typo on the following line:
&gt; {add\_paths,["/home/pi/src/riak-cs-2.1.1/rel/riak-cs/lib/riak\_cs-2.1.1/ebin"$

Your advanced.config file should be similar to this:
&gt; {riak\_kv, [
&gt; 
&gt; {add\_paths,["/home/pi/src/riak-cs-2.1.1/rel/riak-cs/lib/riak\_cs-2.1.1/ebin"]},
&gt; {storage\_backend, riak\_cs\_kv\_multi\_backend},
&gt; {multi\_backend\_prefix\_list, [{&lt;&lt;"0b:"&gt;&gt;, be\_blocks}]},
&gt; {multi\_backend\_default, be\_default},
&gt; {multi\_backend, [
&gt; {be\_default, riak\_kv\_eleveldb\_backend, [
&gt; {data\_root,"/home/pi/src/riak/deps/eleveldb"}
&gt; ]},
&gt; {be\_blocks, riak\_kv\_bitcask\_backend, [
&gt; {data\_root,"/home/pi/src/riak/deps/bitcask"}
&gt; ]}
&gt; ]}
&gt; ]}.


Please let me know if that configuration change gets you going.
