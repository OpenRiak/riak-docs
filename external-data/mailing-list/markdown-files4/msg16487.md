&gt; On Sep 2, 2015, at 11:26 AM, Fred Dushin  wrote:
&gt; 
&gt; I apologize, I was wrong about the timeouts -- they are configurable, either 
&gt; through the client, or in the advanced config on the Riak server(s).
&gt; 
&gt; The timeout gets set in the server here:
&gt; 
&gt; https://github.com/basho/yokozuna/blob/2.1.1/src/yz\_pb\_admin.erl#L114 
&gt; 
&gt; 
&gt; This means you can set the timeout in the PB client, as in
&gt; 
&gt; riakc\_pb\_socket:create\_search\_index(Pid, Index, Schema, [{timeout, Timeout}, 
&gt; ...])
&gt; 
&gt; where timeout is in milliseconds (or the atom 'infinity').
&gt; 
&gt; cf. http://basho.github.io/riak-erlang-client/ 
&gt; 
&gt; 
&gt; The order of precedence is:
&gt; 
&gt; 1. client-defined
&gt; 2. riak config
&gt; 3. default (45 seconds)
&gt; 
&gt; -Fred
&gt;
