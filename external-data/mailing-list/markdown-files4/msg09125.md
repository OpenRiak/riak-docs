&gt; So you are on 12.10, which is one minor release past Precise (12.04) which
&gt; is the current LTS (Long Term Support) release. I haven't confirmed that
&gt; the Precise Riak package will work on Quantal, but I suspect it will be
&gt; fine. In version mismatches most issues you'll find will be on install
&gt; with dependencies. If this is something you will rely on for a business in
&gt; a production sense, I'd recommend using a LTS release like Precise (12.04)
&gt; instead and know that we tested Riak on that platform. If this is
&gt; for development purposes, you will likely be fine on 12.10. If something
&gt; complains on install with dependencies please let me know so I can respond
&gt; differently the next time this comes up on the mailing list.\*\*\*\*
