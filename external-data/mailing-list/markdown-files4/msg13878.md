&gt;&gt; Hi Buri,

&gt;&gt; You upload your XML schema file via an HTTP PUT request:
&gt;&gt;
&gt;&gt; curl -XPUT -iH 'Content-Type: application/xml' '
&gt;&gt; http://localhost:8098/search/schema/my-test-schema' --data-binary
&gt;&gt; @my-test-schema.xml
&gt;&gt;
&gt;&gt; Then, you can associate the schema with an index:
&gt;&gt;
&gt;&gt; curl -XPUT -iH 'Content-Type: application/json'
&gt;&gt; http://localhost:8098/search/index/my-test-index -d
&gt;&gt; '{"schema":"my-test-schema"}'
&gt;&gt;
