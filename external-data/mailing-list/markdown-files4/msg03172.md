The web interface to Riak Search lets you paginate your results using the 
"rows" and "start" query parameters. Here's an example:

http://riakhost:8098/solr/people/select?start=10&rows=10&q=sean

You can find al of this information on the wiki: 
http://wiki.basho.com/Riak-Search---Querying.html#Querying-via-the-Solr-Interface
