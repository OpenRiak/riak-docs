&gt; Hi tele,

&gt; Yes, a secondary index is the most reasonable way to accomplish this.
&gt; Here's an example using the Python client:
&gt;
&gt; now = time.gmtime()
&gt; myobj.add\_index('modified\_int', now)
&gt; myobj.store()
&gt;
&gt; bucket.get\_index('modified\_int', now-3600, now+3600)
&gt;
&gt; Hope that helps.
