On Thu, May 1, 2014 at 11:19 PM, Hector Castro  wrote:

&gt; Hi Chaim,
&gt;
&gt; In order to take advantage of the Riak Search/Solr highlighting
&gt; capabilities from Python (right now), you're best bet may be to use a
&gt; Python Solr client for queries. The Riak client for Python is still not
&gt; 100% compatible with Riak 2.0 (although the master branch has some
&gt; support). [0]
&gt;
&gt; Using the examples provided in the basic usage for search documentation
&gt; [1], I was able to retrieve highlighting snippets via HTTP using the
&gt; following set of query string parameters:
&gt;
&gt; $ curl -s
&gt; "$RIAK\_HOST/search/famous?wt=json&hl=true&hl.fl=name\_s&q=name\_s:Lion\*" |
&gt; python -mjson.tool | grep -A6 "highlighting"
&gt; "highlighting": {
&gt; "default\_yzbucket\_liono\_54": {
&gt; "name\_s": [
&gt; "*Lion-o*"
&gt; ]
&gt; }
&gt; },
&gt;
&gt; (Note the  *tags surrounding the term that matches the query.)
&gt;
&gt; --
&gt; Hector

&gt; [1] http://docs.basho.com/riak/2.0.0beta1/dev/using/search/
