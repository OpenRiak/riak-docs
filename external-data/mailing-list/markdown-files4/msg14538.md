Yokozuna uses actual Solr behind the scenes, so all the caveats that apply
to Solr/Lucene syntax apply to Yokozuna. In this case, no, you cannot use
wildcards at the beginning [1]. As to the minimum length of the stem on a
postfix wildcard, I do not know.

[1] http://lucene.apache.org/core/2\_9\_4/queryparsersyntax.html#Wildcard
Searches
