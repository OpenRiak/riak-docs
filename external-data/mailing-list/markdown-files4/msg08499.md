&gt;&gt; &gt;
&gt;&gt; &gt; JSSourceFunction phaseFunction = new JSSourceFunction(
&gt;&gt; &gt; "function(v) {" +
&gt;&gt; &gt; "return v.sort(function(a, b) {" +
&gt;&gt; &gt; "return a.field - b.field ;" +
&gt;&gt; &gt; "}" +
&gt;&gt; &gt; ");" +
&gt;&gt; &gt; "}");
&gt;&gt; &gt; IndexQuery iq = new BinValueQuery(BinIndex.named("indx\_name"),"bucket",
&gt;&gt; &gt; "value"); // get an index
&gt;&gt; &gt; MapReduceResult execute = client.mapReduce(iq).addMapPhase(new
&gt;&gt; &gt; NamedJSFunction("Riak.mapValuesJson"),
&gt;&gt; &gt; false).addReducePhase(phaseFunction).execute();
&gt;&gt; &gt; String resultRaw = execute.getResultRaw(); // Raw data
&gt;&gt; &gt; Collection result = execute.getResult(YourType .class); //
&gt;&gt; &gt; serialized to a domain object
&gt;&gt; &gt;
&gt;&gt; &gt; The result of this operation will depend on what you return in your
&gt;&gt; &gt; reduce()
&gt;&gt; &gt; function. The result may be a YourType or YourType[] etc. I wanted to
&gt;&gt; &gt; mention that since Jackson can complain about serialization issues if
&gt;&gt; &gt; you
