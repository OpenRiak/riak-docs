This looks to be an issue with the cache used for map reduce. This is a known 
bug and can be mitigated by setting map\_cache\_size to 0 in the riak\_kv section 
of your app.config. Add the following:
{map\_cache\_size, 0},

Grant Schofield
Developer Advocate 
Basho Technologies
