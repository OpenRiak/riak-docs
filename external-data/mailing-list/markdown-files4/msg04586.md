Raghwani,

Although it appears to have multiple entries of the same key, it will only
use the first one it finds (which will be your changed values). Note that
when specifying the default\_bucket\_props in app.config on 0.14.x, you must
also supply the chash\_keyfun as one of the properties:

{chash\_keyfun, {riak\_core\_util,chash\_std\_keyfun}}

This awkwardness will be fixed in an upcoming release.
