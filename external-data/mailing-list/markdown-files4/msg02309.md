&gt;&gt;&gt;&gt;&gt; The choice of "key" as the method/attribute name corresponds to the Riak 
&gt;&gt;&gt;&gt;&gt; terminology for the same concept (and is analogous to "id" in 
&gt;&gt;&gt;&gt;&gt; ActiveRecord). If you want to use a non-String value for the key, create 
&gt;&gt;&gt;&gt;&gt; a property of some other name and then use the "key\_on" method which 
&gt;&gt;&gt;&gt;&gt; makes a defined property act as the key.
&gt;&gt;&gt;&gt;&gt; 
&gt;&gt;&gt;&gt;&gt; property :some\_num, Integer, :presence =&gt; true, :default =&gt; 1234
&gt;&gt;&gt;&gt;&gt; key\_on :some\_num
