&gt; Michael,

&gt; It looks like the second argument to the called function [1]
&gt; (last\_write\_wins), is a binary &lt;&lt;"true"&gt;&gt; instead of the appropriate atom.
&gt; Set the bucket property "last\_write\_wins" to true like so:
&gt;
&gt; curl -X PUT -H "Content-Type: application/json" -d
&gt; '{"props":{"last\_write\_wins":true}}' http://riakhost/buckets/documents/props
&gt;
&gt; Make sure to substitute "riakhost" with whatever host/port combo your Riak
&gt; node is running on.
&gt;
&gt; [1] https://github.com/basho/riak\_kv/blob/1.0/src/riak\_kv\_vnode.erl#L638
