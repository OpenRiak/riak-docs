&gt; similar queries to has\_many :through, but there is no automatic thing for it 
&gt; yet (it needs some more thought). In the meantime, do this:
&gt; 
&gt; class Publisher
&gt; include Ripple::Document
&gt; many :authors
&gt; 
&gt; def books
&gt; authors.map {|a| a.books }.flatten
&gt; end
&gt; end

&gt; On Feb 11, 2011, at 3:39 PM, Mat Ellis wrote:
&gt; 
&gt;&gt; We're still thoroughly in an ActiveRecord mindset over here. Before we go 
&gt;&gt; and build some simple collect statements to replace a few has\_many :through 
&gt;&gt; statements, is there some Ripple or Riak goodness we should be using instead?
&gt;&gt; 
&gt;&gt; Example:
&gt;&gt; 
&gt;&gt; \* A Publisher has many Authors who have many Books
&gt;&gt; \* We want to get a set of books for a given publisher
&gt;&gt; \* In ActiveRecord we would say:
&gt;&gt; 
&gt;&gt; # publisher.rb
&gt;&gt; class Publisher &lt; ActiveRecord::Base
&gt;&gt; has\_many :authors
&gt;&gt; has\_many :books, :through =&gt; authors
&gt;&gt; end
&gt;&gt; 
&gt;&gt; # author.rb
&gt;&gt; class Author &lt; ActiveRecord::Base
&gt;&gt; belongs\_to :publisher
&gt;&gt; has\_many :books
&gt;&gt; end
&gt;&gt; 
&gt;&gt; # book.rb
&gt;&gt; class Book &lt; ActiveRecord::Base
&gt;&gt; belongs\_to :author
&gt;&gt; end
