For what it's worth, it looks good on the packagecloud directions as
of the most recent update [1]:

 &gt; curl -s 
 &gt; https://packagecloud.io/install/repositories/basho/riak-cs/script.deb.sh
| sudo bash
 &gt; sudo apt-get install riak-cs=2.0.1-1

[1] 
https://packagecloud.io/basho/riak-cs/packages/ubuntu/precise/riak-cs\_2.0.1-1\_amd64.deb
