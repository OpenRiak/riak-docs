Like I said, you seem to be using R15 in your building of riak, that's why
you have a erts-5.9 directory instead of erts-5.8.4. Please uninstall
your erlang version, and try using erlang-r14b03. Using something like
kerl  will make it easier to manage
multiple versions of erlang.

Good luck,
Jared
