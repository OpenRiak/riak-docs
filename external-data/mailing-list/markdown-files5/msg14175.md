Found it. The reason was I had created an index without a schema:

 iriak:create\\_search\\_index(&lt;&lt;"index\\_name"&gt;&gt;).

but it should be created with custom schema:

 iriak:create\\_search\\_index(&lt;&lt;"index\\_name"&gt;&gt;,&lt;&lt;"schema\\_name"&gt;&gt;).
