&gt; Hi Jurgen,
&gt;
&gt; When a node is started for the first time, its name is recorded in the
&gt; ring file (a file under /var/lib/riak/ring by default on CentOS). For a
&gt; single node that hasn't joined a cluster, yet, the easiest way to start the
&gt; node with a new node name is to simply remove the ring directory and then
&gt; start the node. Please note that any changes to bucket properties for
&gt; buckets under the default bucket type get lost in this process. If any such
&gt; changes have been made, re-apply them once the node has been restarted.
