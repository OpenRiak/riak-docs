&gt;&gt; I believe they actually recommend adding all the nodes you want to add in
&gt;&gt; a
&gt;&gt; single step, instead of one by one - this way the hand-off will happen
&gt;&gt; only
&gt;&gt; once, and all the objects will go to the right nodes immediately.
&gt;&gt;
&gt;&gt; If you add nodes one by one, the objects will have to do multiple
&gt;&gt; transitions, since each added node will cause a redistribution of which
&gt;&gt; nodes will be responsible for which objects, causing a hand-off process
&gt;&gt; each
&gt;&gt; time, and you will consume more traffic, time and more resources to get
&gt;&gt; the
&gt;&gt; same final result.
&gt;&gt;
&gt;&gt;
