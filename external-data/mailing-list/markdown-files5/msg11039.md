&gt;&gt; Sharing a tip with the list, in case it saves a future person an hour of
&gt;&gt; their time..
&gt;&gt;
&gt;&gt; If you use Nginx to loadbalance to Riak CS, be sure that you set this in
&gt;&gt; your site configuration:
&gt;&gt;
&gt;&gt; proxy\\_set\\_header Host $http\\_host;
&gt;&gt;
&gt;&gt; This ensures that the HTTP Host: header is passed through as it is
&gt;&gt; received, rather than replaced by the IP address or hostname of the back-end
&gt;&gt; server that nginx proxied the request to.
&gt;&gt;
&gt;&gt; Also ensure that the domain name fully matches the one in
&gt;&gt; /etc/riak-cs/app.config, and that you have wildcard DNS records configured
&gt;&gt; against it.
&gt;&gt;
