&gt; Installing erlang doesn't require sudo either... just --prefix=~/erlang
&gt; ie. git clone git://github.com/erlang/otp
&gt; git checkout OTP\\_R15B0?
&gt; ./otp\\_build autoconf
&gt; ./configure --prefix=~/erlang --libdir=~/erlang/lib64 --enable-m64-build
&gt; --enable-hipe --enable-smp-support --enable-threads --enable-kernel-poll &&
&gt; make && make install
&gt;
&gt; PATH=$PATH:~/erlang/lib64/erlang/bin
&gt;
&gt; tada you're a riak kungfu master..
&gt; erlang also doesn't require any ports &lt; 1024
