Hi Marcel,

You can change the number of allowed transfers via the riak-admin tool:

$ riak-admin transfer-limit
=============================== Transfer Limit
================================
Limit Node
-------------------------------------------------------------------------------
 2 'riak@127.0.0.1'
-------------------------------------------------------------------------------
Note: You can change transfer limits with 'riak-admin transfer\\_limit
'
 and 'riak-admin transfer\\_limit  '
