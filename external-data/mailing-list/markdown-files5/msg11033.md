On 25.04.2013, at 20:16, Dmitri Zagidulin  wrote:

&gt; \\* Disk IO speed is very important, and is usually the bottleneck for 
&gt; clusters. RAID or SSDs are helpful. If you're running on AWS, disk IO is 
&gt; especially killer, so spring for sufficient Provisioned IOPS.

Keep in mind that PIOPS offer provisioned IOPS they have an issue over using 
the ephemeral storage which is higher latency. If you care about the &gt;99% 
latencies, you are most of the time better of by using the ephemeral storage. 
If you just care about throughput, PIOPS is for you. But, as always, YMMV.\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_\\_
