&gt;&gt; You're getting a runtime load error because your are likely not linking
&gt;&gt; against the right C++ runtime (because you built with gcc). IIRC the GCC
&gt;&gt; C++ runtime and the C++ runtime that ships with Darwin are not ABI
&gt;&gt; compatible. Welcome to C++.
&gt;&gt;
&gt;&gt; You might try hacking your environment using DYLD\\_LIBRARY\\_PATH, so that
&gt;&gt; you link the gcc runtime.
&gt;&gt;
&gt;&gt; Some things you can do to help diagnose:
&gt;&gt;
&gt;&gt; See what your eleveldb is linked against.
&gt;&gt;
&gt;&gt; otool -L
&gt;&gt; /Users/Humberto/riak-2.1.1/rel/riak/bin/../lib/eleveldb-2.1.0-0-ga36dbd6/priv/eleveldb.so
