&gt; Hi Syed. I can't comment upon UGameDB but generally speaking you will need
&gt; to set a content-type header indicating the mime type. For a JPEG image
&gt; that type would be image/jpeg for a PNG it would be image/png. Riak just
&gt; regards data as opaque blobs so it is probably defaulting to text/plain.
&gt; Take a look at the examples on the docs website -
&gt; http://docs.basho.com/riak/latest/dev/using/basics/
