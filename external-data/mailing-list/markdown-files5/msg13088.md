Greetings and salutations. Today Basho is pleased to bring you an updated 
version of Riak CS, our cloud storage solution.

Highlights:
Performance improvements to accommodate workloads where the same object (same 
key) is rapidly overwritten.
Incorporates Lager 2.0.1 to address a recent tracing bug.
Fixes a bug where an upload that experiences corruption (or at least has a 
faulty MD5 checksum) will both reject the upload (correct behavior) and remove 
any older version of the object (distinctly incorrect behavior).
Release notes: 
http://docs.basho.com/riakcs/latest/cookbooks/Riak-CS-Release-Notes/
Downloads: http://docs.basho.com/riakcs/latest/riakcs-downloads/

Thanks.
