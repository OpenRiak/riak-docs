&gt; Riak Java Client 1.1.2 and 1.4.0
&gt; Via Nabble - Riak Users
&gt; Greetings!
&gt;
&gt; The Riak Java client versions 1.1.2 and 1.4.0 have been released and
&gt; are now available from maven central. For non-maven users a bundled
&gt; jar including all dependencies can be found for these versions at:
&gt;
&gt; http://riak-java-client.s3.amazonaws.com/riak-client-1.4.0-jar-with-dependencies.jar
&gt; http://riak-java-client.s3.amazonaws.com/riak-client-1.1.2-jar-with-dependencies.jar
&gt;
&gt; Javadoc is available via: http://basho.github.io/riak-java-client/
&gt;
&gt; Why two versions you ask? Good question.
&gt;
&gt; The recently released Riak 1.4.0 adds a number of new features and
&gt; brings parity between our HTTP and Protocol Buffers APIs. The Java
&gt; client 1.4.0 reflects this by allowing PB operations that previously
&gt; would throw exceptions (setting bucket properties, for example) and
&gt; supports those new features. If you're using Riak 1.4, you want to be
&gt; using the Java client 1.4.x to use the new features.
&gt;
&gt; 1.1.2 on the other hand is a minor maintenance / bug fix release for
&gt; use with Riak 1.3.x and below. It will also work with Riak 1.4 but
&gt; does not support the new Riak 1.4 features.
&gt;
&gt; Probably the most notable feature in 1.4.0 is support for the new
&gt; counters in Riak. Check out the Javadoc here:
&gt; http://basho.github.io/riak-java-client/1.4.0/com/basho/riak/client/operations/CounterObject.html
&gt;
&gt; Thanks!
