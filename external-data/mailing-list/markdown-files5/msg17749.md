Hi Arun -

When you install Riak it installs the Erlang VM to a well-known location,
like /usr/lib/riak/erts-5.9.1

You can use /usr/lib/riak/erts-5.9.1/bin/erlc and know that it is the same
Erlang that Riak is using.
