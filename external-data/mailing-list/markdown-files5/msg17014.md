Hi all,

i want to setup a Riak DB such that:

all nodes have one replica of the data, so that any single node can work on 
it’s own in case of network failures.

I’m intending to reach that goal by:

- Setting n\\_val equal number of nodes, e.g. 4
- Making sure that the Ring Size (e.g. 64) can be divided evenly by the 
number of nodes

Is it correct that in this case all the data will always be distributed evenly 
on all the nodes, that means each node will have a copy of all the data?
