&gt; https://github.com/basho/yokozuna/blob/develop/priv/default\\_schema.xml#L124-L131

&gt; And these:
&gt; https://github.com/basho/yokozuna/blob/develop/priv/default\\_schema.xml#L101-L104
&gt; -
&gt; These dynamic fields catch all Riak DTs because the solr field names of
&gt; data types automatically get their type name appended to the end (as you
&gt; noticed with your reference to "\\*likes\\_counter" \\*in your own index).
&gt;
&gt; As you can see in the default schema, all sets are automatically indexed
&gt; as multivalued.
