&gt; Hi,
&gt; That warning means that riak failed to calculate the leveldb read block
&gt; error count stat.
&gt;
&gt; This is caused by a bug fixed in 1.3.2. The stat code picks a random vnode
&gt; from 1 to num\\_partitions on the node and asks it for the read block error
&gt; stat. If your node has 1 or fewer partitions this error occurs.
&gt;
&gt; It is an edge case, but one that people seem to hit from time to time. As
&gt; I said, fixed for 1.3.2 and .1.4, as far as I know: what version of Riak
&gt; are you running when you see this?
&gt;
&gt; Cheers
&gt;
&gt; Russell
