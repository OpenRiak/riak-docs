&gt;&gt; Yes, a secondary index is the most reasonable way to accomplish this.
&gt;&gt; Here's an example using the Python client:
&gt;&gt;
&gt;&gt; now = time.gmtime()
&gt;&gt; myobj.add\\_index('modified\\_int', now)
&gt;&gt; myobj.store()
&gt;&gt;
&gt;&gt; bucket.get\\_index('modified\\_int', now-3600, now+3600)
&gt;&gt;
