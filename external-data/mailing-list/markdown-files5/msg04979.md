在 2011年10月1日 下午11:15，Ian Plosker 写道：

&gt; There are instructions on the wiki:
&gt; http://wiki.basho.com/Secondary-Indexes.html#Migrating-an-Existing-Cluster-(Method-One)
&gt;
&gt; Basically, you will have each node leave the cluster, switch the backend,
&gt; and then rejoin. The process is something like this:
&gt;
&gt; 1) Issue `riak-admin leave` and wait for all transfers to complete
&gt; 2) `riak stop`
&gt; 3) Change the backend to the `riak\\_kv\\_eleveldb\\_backend`
&gt; 4) `riak start` and `riak-admin join `
&gt; 5) Repeat for each node in the cluster
&gt;
&gt;
