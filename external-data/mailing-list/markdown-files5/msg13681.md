Abhishek,

You're conflating two completely different version of Riak Search. If you 
downloaded Riak pre11, you need to reference the pre11 documentation:

http://docs.basho.com/riak/2.0.0pre11/dev/using/search/

search-cmd is unusable for 2.0 Search. You no longer manually manage indexes in 
new Riak Search, but instead, you associate a bucket with an index, and then 
insert values into Riak. It will manage indexing for you.

Eric

&gt; Hi Abhishek,

&gt; The Riak Python client does not yet have official support for the
&gt; version of Riak Search in Riak 2.0.0pre11. That said, you should be
&gt; able to add values to a Riak 2.0 cluster (with search configured and
&gt; enabled) and then query those values using a Python Solr client.
&gt; 
&gt; In addition, to enable search in Riak 2.0.0pre11, you need to edit the
&gt; `riak.conf` configuration file. Inside `riak.conf` look for `search =
&gt; off` and switch that to `search = on`. After that, please see the Riak
&gt; Search simple setup section in our online documentation for further
&gt; details on indexing and querying data. [0]
&gt; 
&gt; --
&gt; Hector
&gt; 
&gt; [0] http://docs.basho.com/riak/2.0.0pre11/dev/using/search/#Simple-Setup
