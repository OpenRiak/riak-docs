Hi,
We turned on Github issues for the riak-java-client last night. This is another 
way to contribute to the riak-java-client. Basho's bugzilla[1] will still be 
the canonical repository of bug information. Until we release 1.0, I will be 
taking care of manually syncing the github issues with bugzilla, but it will be 
\\*one way\\* from GH-&gt;BZ. After 1.0 I'll work out a way to keep all issues visible 
in both systems.

Hopefully this will make it easier for anyone so minded to contribute to 
riak-java-client.

Have at it!

Russell
