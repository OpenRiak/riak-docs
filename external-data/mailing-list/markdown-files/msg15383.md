Riak Users,

Riak CS 1.5.3 has been released and can be downloaded from the
downloads page[1]. This release contains a fix for storage
calculations and updates to avoid sibling explosion and add
configurable timeouts for Riak interactions. Please review the Release
Notes [2] for more details before upgrading.

[1] http://docs.basho.com/riakcs/latest/riakcs-downloads/ [2]
