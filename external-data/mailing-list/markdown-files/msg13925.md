Dear Riak Community,

We’re pleased to let you know we’ve pushed an updated preview build, 2.0pre20, 
to our downloads[1] site. Please note that although we’re nearing beta, this is 
still a preview build.

Known Issues:
If you use the Data Types API without "include\_context=false" Maps and Sets 
will not work correctly.
