PraveenKumar,

You can set those parameters in the rebar.config file of basho\_bench.

For example if you wanted to set the maximum number of processes to 65536 you 
would do:

{escript\_emu\_args, "%%! +P 65536 \n"}.

Hope that helps,
Jared
