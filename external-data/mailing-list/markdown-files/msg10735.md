The instructions page for "Configuring Riak for CS" at

http://docs.basho.com/riakcs/latest/cookbooks/configuration/Configuring-Riak/

contains some erroneous information.

In the instructions to configure riak for a multi backend scenario, it says
to modify /etc/riak/app.config with

{storage\_backend, riak\_cs\_kv\_multi\_backend}

which I believe should read

{storage\_backend, riak\_kv\_multi\_backend}
