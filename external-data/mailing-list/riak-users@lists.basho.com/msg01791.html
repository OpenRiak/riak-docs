On Tue, Dec 14, 2010 at 2:22 PM, Sean Cribbs <s...@basho.com> wrote:
Carson,
Unfortunately the PB interface, as you discovered, doesn't support all the bucket properties. You could, however, also use the riak- erlang-http-client which has full support for all bucket properties. We realize this is a pain point and will be working to improve the PB interface in the near future. To enable the precommit hook, do something like this with the HTTP client:
Client = rhc:create(), %% use create/4 if you're not connecting to localhost 
rhc:set_bucket(Client, "my_bucket", [{<<"precommit">>, [
{struct, [{<<"mod">>, <<"riak_search_kv_hook">>}, {<<"fun">>, <<"precommit">>}]} 
   ]}]).
