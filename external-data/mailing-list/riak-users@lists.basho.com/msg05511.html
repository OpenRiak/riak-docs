Hi Jon,

I did the following to solve the OTP release issue with Homebrew:

        cd /usr/local
        # Checkout commit right before r14b04 release
        git checkout -b r14b03 9332ca98df235cf3b375ee490065947f19c6a3c8
        brew install erlang
        git checkout master

--
Hector
