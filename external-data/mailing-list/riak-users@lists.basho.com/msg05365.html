Re: Riak 1.x with Secondary Indexes

Jeremiah Peschka Sat, 29 Oct 2011 08:46:15 -0700 

The protocol buffers API already makes it easy to store the indexes. Querying 
them is annoying, but there's an Erlang client method to query an index that's 
pretty easy to use. Have a look at the get_index functions in 
riakc_pb_socket.erl. They're around line 700 or so.
