Re: Why does ripple cloned object can change the state of original object?

Sean Cribbs Thu, 24 Feb 2011 07:19:47 -0800 

Ripple::Document does not support cloning at the moment.  Note that 
Object#clone simply copies the instance variables but doesn't clone them 
(unless the class overrides #initialize_copy to change that behavior).
