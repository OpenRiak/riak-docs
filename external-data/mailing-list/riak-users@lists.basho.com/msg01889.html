Hello,

I have three-node riak search cluster, with buckets enabled for indexing and
the apropriate schema for the index

Using riak http client the document is inserted with success.

365> {ok, Pid1} = riakc_pb_socket:start_link(Server, 8087).

           {dict,1,16,16,8,80,48,
                 {[],[],[],[],[],[],[],[],[],[],[],[],[],[],[],...},
                
{{[],[],[],[],[],[],[],[],[],[],[[<<...>>|...]],[],[],...}}},
           <<131,108,0,0,0,113,97,123,107,0,11,34,100,111,109,97,
             105,110,95,105,100,34,...>>}
367> riakc_pb_socket:put(Pid1, Obj).
{error,<<"{precommit_fail,{error,missing_field,<<\"value\">>}}">>}
