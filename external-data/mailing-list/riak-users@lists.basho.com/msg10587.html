Recap for March 18 - March 25

Adron Hall Mon, 25 Mar 2013 11:36:26 -0700 

All - here's the latest in the land of Riak. Big news, including a week
close to RICON East & Riak CS is now Apache 2.0 Open Source! A native OS-X
Browser for Riak just got committed by Ilja, Scala Client updates & more.

RICON East Updates:

Several talk details landed on the RICON East site over the weekend:
- http://ricon.io/east.html#zookeeper-for-the-skeptical-architect
- http://ricon.io/east.html#nobody-ever-got-fired-for-picking-java
- http://ricon.io/east.html#just-open-a-socket
- http://ricon.io/east.html#call-me-maybe
- http://ricon.io/east.html#realtime-systems
- http://ricon.io/east.html#high-availability-with-riak-and-postgresql
- http://ricon.io/east.html#large-scale-data-service-as-a-service
- http://ricon.io/east.html#optimizing-leveldb

Tickets are still available: http://ricon-east-2013.eventbrite.com/

On Wednesday, May 20 Basho open sourced Riak CS ->
- http://basho.com/riak-cs-is-now-open-source/

Riak CS updated docs & more ->
- http://docs.basho.com/riakcs/latest/

Basho hustlers Tom Santero and Hector Castro will be in Boston this week
for Monitorama. Find them and ask them hard questions. They're also hosting
a Riak Meetup at Basho HQ the night before, so if you're in town you should
stop by. ->
- http://www.meetup.com/Boston-Riak/events/108304182/

Andy Gross will be presenting on Riak CS at the upcoming Citrix Cloud
Computing Meetup.
- http://www.meetup.com/cloudcomputing/events/101130082/

Blog post: Using Riak to understand how Erlang applications are packaged
and started
-
http://ntcoding.blogspot.co.uk/2013/03/using-riak-to-understand-how-erlang.html

riak-scala-client 0.8.0 released
- http://riak.scalapenos.com/
- https://github.com/agemooij/riak-scala-client

Jonathan Perkin wrote a blog post about Installing SVR4 packages on SmartOS
and highlighted Riak.
- http://www.perkin.org.uk/posts/installing-svr4-packages-on-smartos.html

OS-X Native Riak Browser just got committed on Github via Ilja Iwas.
- https://github.com/iljaiwas/RiakBrowser

Riak will have excellent representation at the upcoming NoSQL Matters
Conference.
- http://2013.nosql-matters.org/cgn/agenda/

Congratulations are in order for to Michael Bernstein for merging the the
first PR to his Riak Go client yesterday. Update to protocol buffer
compiled code and related updates reenabling compilation. Details ->
- https://github.com/mrb/riakpbc/pull/1
