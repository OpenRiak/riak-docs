Re: Hit a snag in using the MultiPartUpload

Kelly McLaughlin Tue, 27 Aug 2013 15:10:50 -0700

Hi Shannon. I think the problem you are having is that you are trying to use the POST method to upload a part when you should be using PUT. Riak CS interprets a POST request that includes the uploadId like you are sending to be an attempt to complete the multipart upload and that is why you are seeing that error. I think if you use PUT to upload the parts it should work fine for you.

Kelly
