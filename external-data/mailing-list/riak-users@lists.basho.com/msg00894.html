Dan Reverri Mon, 16 Aug 2010 06:28:21 -0700 

Hi Senthilkumar,

You can set bucket properties in two ways.

1. Set the default bucket properties in app.config which will be used by
buckets which have not been explicitly defined.

2. Set the bucket properties through the REST API.
https://wiki.basho.com/display/RIAK/REST+API#RESTAPI-Setbucketproperties

You can set bucket properties through the REST API without putting documents
in the bucket.


You'll have to determine which method to use based on your application's
requirements.
