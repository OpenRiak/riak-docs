Junk, Damion A Wed, 12 Oct 2016 09:19:26 -0700 

Hello all -

I was wondering if it is possible to use RiakTS with a multi-backend 
configuration. 

I have an existing set of applications using RiakKV and Bitcask, but we're now 
wanting to start using some of the TS features on a new project. Setting up the 
multi-backend configuration with TS seems to work fine (our app existing app 
reads the Bitcask buckets and SOLR indices without issue), and we can even 
create a TS schema. We have LevelDB as the default backend, so presumably this 
is what was used during the "create table" query.

Client).
