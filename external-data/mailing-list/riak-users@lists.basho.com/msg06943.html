Re: Can't find docs/tutorial on secondary indexes for riak-erlang-client

Daniil Churikov Mon, 12 Mar 2012 05:53:45 -0700 

Seems that i discover how to use this 2i range queries from riak_client:

> {ok, C} = riak:local_client(),
> C:get_index(<<"bucket_name">>, {range, <<"$key">>, <<"k1">>, <<"k7">>}).
> {ok, [<<"k7">>, <<"k6">>, <<"k5">>, <<"k4">>, <<"k3">>, <<"k2">>,
> <<"k1">>]}

where <<"$key">> is a special value ofr key index.
I test from shell such way:
$ curl -v -X GET
'http://localhost:8098/buckets/bucket_name/index/$key/k1/k7'
