We are creating a lot of buckets because I needed the extra namespace.  My 
alternative was constantly updating a value, which in some cases would grow the 
value very large.  I figured keeping the key/value groups small would be quite 
a bit faster, and since the docs said you could have as many buckets as you 
like I thought I was good.  I just didn't play close enough attention to the 
"except when" clause…:)

Thanks for the help everyone.  I'll have some more questions soon.  I ran into 
some challenges over the weekend that I want to learn more about…

>>>> For those interested, I identified my performance problem.

>>>> I was creating a lot of buckets, and the properties did not match the
