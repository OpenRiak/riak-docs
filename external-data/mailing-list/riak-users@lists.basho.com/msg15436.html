Wiping Riak 2.0 data

Cosmin Marginean Sun, 04 Jan 2015 04:25:54 -0800 

Hi,  

I’m running Riak 2.0.2 on Centos 6.5 installed from an RPM 
(http://s3.amazonaws.com/downloads.basho.com/riak/2.0/2.0.2/rhel/6/riak-2.0.2-1.el6.x86_64.rpm)

I’m interested in wiping the entire storage (in order to re-run tests, etc). I 
have a script that traverses buckets/keys but I’m aware that this process is 
costly, plus I might be dealing with some eventual-consistency issues.

Consequently, I would prefer to wipe the “data” directory entirely if that’s in 
any way possible, and re-start Riak. However, the structure seems to be 
scattered in several directories and is not entirely clear which one needs to 
be wiped for a “clean data reset” without corrupting anything:

# ll /var/lib/riak/
