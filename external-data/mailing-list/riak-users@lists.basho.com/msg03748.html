Re: Riak crash on 0.14.2 riak_kv_stat terminating

Sean Cribbs Sun, 19 Jun 2011 13:52:39 -0700 

Jeremy,

There is an open pull request for fixing this bug.  Details of the bug are
here: https://issues.basho.com/show_bug.cgi?id=1072  It will not cause your
entire to node to crash, there is likely some other cause. Node exits tend
to only happen when you run out of RAM or disk space, or can't bind to a TCP
port during riak_core startup. Were there other indications in the
erlang.log.*?
