Riak KV 2.9.0 released at Code BEAM STO 2019

Nicholas Adams Fri, 17 May 2019 08:06:00 -0700 

Dear All,
I am extremely pleased to announce with Martin Sumner at Code BEAM STO 2019 
that Riak KV 2.9.0 has officially been released!

GitHub
https://github.com/basho/riak/tree/riak-2.9.0

Packages
https://files.tiot.jp/riak/kv/2.9/2.9.0/

Many thanks to everybody who contributed to this great achievement.

As always any issues or questions, please post to GitHub, this mailing list or 
the Slack channel.
