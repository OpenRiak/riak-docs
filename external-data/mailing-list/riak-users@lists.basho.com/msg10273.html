Re: Is per-bucket auto-expiry possible?

Kresten Krab Thorup Wed, 27 Feb 2013 02:41:07 -0800 

You can configure separate backends with varying auto-expire (using 
multi_backend), and then define a per-bucket property which decides which 
backend its objects are stored in.  That's as good as it gets right now.

Kresten
