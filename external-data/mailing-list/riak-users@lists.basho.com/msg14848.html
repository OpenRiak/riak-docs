> > > - Hit CTRL+C twice to quit from the node
> > > 
> > > You can then revert back to the normal `info` logging level by running 
> > > the following command via `riak attach`:
> > > 
> > > - `riak attach`
> > > - `(riak@nodename)1> SetInfo = fun() -> {node(), 
> > > lager:set_loglevel(lager_file_backend, "/var/log/riak/console.log",

> > > - `(riak@nodename)2> rp(rpc:multicall(erlang, apply, [SetInfo,[]])).`
