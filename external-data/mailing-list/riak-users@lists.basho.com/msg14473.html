Naveen -

The current master branch on the riak-python-client repo targets Riak 2.0, not 
Riak 1.4.x any more.

The latest released client is 2.0.3 (not be be confused with Riak 2.0) which is 
available on PyPI https://pypi.python.org/pypi/riak/2.0.3 or on GitHub with tag 
“2.0.3” https://github.com/basho/riak-python-client/releases/tag/2.0.3.  This 
has the BadStatusLine fix already in it.

Brett
