Rebar requires version 1.1 or higher of Mercurial. What version do you have installed? You can find out using 'hg --version' 

jons-macpro:riak jmeredith$ hg --version
Mercurial Distributed SCM (version 1.5.1)

If you are below 1.1 you will need to upgrade.

--Jon.
