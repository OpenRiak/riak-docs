Apologies, a stray click sent this before it was ready.

There is a known issue with building from source with pre-R15 erlang,
however all the packages use R15B and do not suffer from it.  A fix has
been committed to the 1.2 branch, so please pull directly from there if
needed.

Jon

>

> app](https://github.com/basho/riak_core/pull/217)
>

> https://github.com/basho/riak_core/issues/166)
>
>
> * [riak_core - Make the ring manager responsible for loading the ring](
> https://github.com/basho/riak_core/pull/214)
>
> * [riak_core - Fix capability system race condition](
> https://github.com/basho/riak_core/pull/216)
>
>
> * [riak_kv - Changed semantics of backend:drop - backend must close all
> handles](https://github.com/basho/riak_kv/pull/373)
>
> * [riak_kv - Call eleveldb:close on vnode stop for eleveldb backend](
> https://github.com/basho/riak_kv/pull/372)
