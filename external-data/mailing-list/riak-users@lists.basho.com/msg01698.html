Re: Installed Riak via ssh. Now can/should I test *remotely* from a browser?

Sean Cribbs Thu, 02 Dec 2010 05:19:07 -0800 

Be aware that nginx's reverse-proxy configuration may not fully support HTTP 
1.1 semantics.  If you need those semantics, you might be better off using a 
dedicated reverse proxy like haproxy or pound.
