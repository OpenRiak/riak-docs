H. Ibrahim YILMAZ Thu, 28 Mar 2013 09:43:38 -0700 

I'm trying to work with Riak on PHP. I read the documentation but no luck.
So I've already configured the "search" option in /etc/init.d/app.conf as true. 

So, print_r($results) returns as null array.

<?php
require_once('riakclient.php');
$client = new RiakClient('127.0.0.1', 8098);

$bucket = $client->bucket("searchbucket");

$bucket->newObject("two", array("foo"=>"two", "bar"=>"green"))->store();

$results = $client->search("searchbucket", "foo:one")->run();
