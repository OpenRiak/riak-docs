Try running `riak-admin top` on the node to see what's going on:

http://docs.basho.com/riak/1.4.12/ops/running/tools/riak-admin/#top 
<http://docs.basho.com/riak/1.4.12/ops/running/tools/riak-admin/#top>

This may give you some insight into what that node is doing.
