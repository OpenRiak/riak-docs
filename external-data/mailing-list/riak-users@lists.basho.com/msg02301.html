>>>>>> The choice of "key" as the method/attribute name corresponds to the Riak 
>>>>>> terminology for the same concept (and is analogous to "id" in 
>>>>>> ActiveRecord).  If you want to use a non-String value for the key, 
>>>>>> create a property of some other name and then use the "key_on" method 
>>>>>> which makes a defined property act as the key.
>>>>>> 
>>>>>> property :some_num, Integer, :presence => true, :default => 1234
>>>>>> key_on :some_num
