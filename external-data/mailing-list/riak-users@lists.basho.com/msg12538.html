Steve Vinoski Mon, 07 Oct 2013 17:06:26 -0700 

As indicated in the URL for the tarball you downloaded, Riak 1.4.2 for OS X
was built for 10.8.x, but you're running 10.7.5 so that build won't work
there.

--steve
