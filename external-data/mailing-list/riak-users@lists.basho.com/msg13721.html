But it seems happened on my new Debian 7.3(X86_64) test box with

Without those lines,

-proto_dist inet_tls
-ssl_dist_opt client_certfile "{{platform_etc_dir}}/erlclient.pem"

With such configuration, I got riak started failed but riak process running 
with errors as following(in the end of this mail).

Any idea?
