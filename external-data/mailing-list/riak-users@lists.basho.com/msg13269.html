Riak Recap for December 10 - 17
========================

Philipp Fehre posted slides from his Riak 2.0 talk at the Riak user
group in Munich.
- 
http://sideshowcoder.com/post/69778936838/what-is-new-in-riak-2-0-at-riak-ug-munich

Covata shipped Safe Share, which is a corporate dropbox that leverages Riak.
- http://basho.com/covata-enables-self-defending-data-with-the-help-of-riak/

Basho's own Matthew Revell gave a talk on Riak at NoSQL Matters Cologne.
- http://www.youtube.com/watch?v=z7aahZ8w4Ro

The video of Garrett Eardley's standing-room only talk at Strange Loop
2013 on how Riot Games uses Riak is now online.
- http://www.infoq.com/presentations/riak-game-scale

A recording of last week's Riak community hangout on Java and C
clients is available.
- http://www.youtube.com/watch?v=Dpf9OqO-wK8 - RC hangout 004

Noah Isaacson posted a Dockerfile to create a Riak container to the
Docker Index.
- https://index.docker.io/u/nisaacson/riak/

Hector Castro hacked on a Vagrant project over the weekend to setup a
local Riak backed Datomic instance.
- https://github.com/hectcastro/vagrant-datomic-riak
