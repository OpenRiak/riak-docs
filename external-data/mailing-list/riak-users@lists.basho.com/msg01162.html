Re: bitcask 42?

Alexander Sicular Thu, 30 Sep 2010 12:02:58 -0700

The default vnode number is 64. If you are connecting 63 physical nodes togeter in one riak cluster than you really want to increase the number of vnodes. There is some recommended metric for physical nodes to vnodes. I think it's like a minimum of 4 vnodes per physical node. But I could be totally making that up.
