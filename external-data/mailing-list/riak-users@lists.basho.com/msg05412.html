Hi,

http://wiki.basho.com/Vector-Clocks.html discusses allow_mult=true but does 
not really explain how allow_mult=false works, at least not when 
last_write_wins is false.

Here's a table we can fill out:

allow_mult, last_write_wins
-----------
false, false -> ?
true, false -> write siblings, read siblings
false, true -> writes overwrite existing, read single values
true, true -> invalid
