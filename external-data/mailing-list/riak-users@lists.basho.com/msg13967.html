Hello!

I have 8x cluster of riak+riak-cs on debian. Config templates attached
Versions:
ii  riak                            1.4.8-1
amd64        Riak is a distributed data store
ii  riak-cs                         1.4.5-1
amd64        Riak CS

frontend (Tengine version: Tengine/1.5.1 (nginx/1.2.9)), config
attached
