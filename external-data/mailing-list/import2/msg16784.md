Hi Victor,

Riak ships with OTP compiled from this tag:
https://github.com/basho/otp/tree/OTP\_R16B02\_basho8

I believe the next version of Riak will use the OTP\_R16B02\_basho9 tag.

I doubt Riak will compile successfully using version 18. I can say that
Riak has not been tested with that version.
