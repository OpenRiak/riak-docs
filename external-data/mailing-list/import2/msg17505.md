Riak Users,

Riak 2.0.7 was released on June 27th and can be downloaded from the
downloads page[1]. This release contains new updates to batching and
Solr, Cuttlefish configurations, an OTP upgrade, and fixes for two
product advisories[2]. Please review the Release Notes [3] for more
details before upgrading.
