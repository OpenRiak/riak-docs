Here is your biweekly Recap! It's been a busy few weeks. There is more to
see on the mailing list.

## Announcements
\* Justin Plock released SmokeTurner's Notification service. It is a great
example of a Java service written on top of Riak KV [0]
\* We announced our Go client (Golang) arriving as a Beta, which started
thanks to Timo Gatsonides work [1]
\* The .NET client received an update to 2.1.1 to address a connection
condition in RiakPbcSocket [2]

## Recently Answered
\* Anthony has an out-of-balance Riak S2 (aka Riak CS) cluster that Arnaud
helps explore [3]
\* Christopher Meiklejohn helped Humberto find his compiler error [4]
\* MVM delves into leveldb functionality [5]
\* Henrik had some help thinking through replacing AWS S3 with Riak S2 (aka
Riak CS) [6]
\* Russell helps Hao understand the flag types available in the Erlang
client [7]
\* Alex and Cuyler explain the open file limit setting [8]
\* Dmitri gives a thorough answer to a 'max number of buckets' question [9]
\* Drew explains the Apache Solr extraction functionality within Riak Search
[10]

## Open Questions - Need Your Help
\* Paulo sees an error every hour on his 2.1.1 development cluster [11]
\* Amao sees an outbound handoff error and still needs some help [12]
\* Humberto has trouble with riak\_erlang\_client while writing an extension
to riak\_pb [13]
\* JCesar is curious how to best use Spark with Riak KV [14]

## What'd we miss?
Have you brought Riak into production at a new place? Did you find a blog
or presentation helpful? Good! Share it in our release notes:
https://github.com/basho-labs/the-riak-community/tree/master/release-notes

Have a great weekend,
