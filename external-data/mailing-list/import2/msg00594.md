> Hi Simon,
>
> Static arguments can be passed to MapReduce queries using the "arg"
> property. Here is an example query that runs over all the keys in the bucket
> "bucket":
> {
> "inputs": "bucket",
> "query": [
> {
> "map": {
> "language": "javascript",
> "source": "function(value, keyData, arg) { return [arg];
> }",
> "keep": true,
> "arg": "static data"
> }
> }
> ]
> }
