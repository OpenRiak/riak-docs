To be clear, this patch adds the ability for nodes that have multiple backends 
(some buckets are leveldb, some are bitcask, etc.)
to support 2i on the buckets that use the leveldb backend. This does not add 
support for 2i for buckets that use bitcask (or any other
non leveldb backend).
