For each message file:

1. Extract:
- Author (name only)
- Date (Thu, 14 Apr 2016 07:06:15 -0700 -> YYYY-MM-DDTHH:mm:SS-00:00 ie UTC)
- Title/Subject (text)
- Content (long text)
- Thread structure - flatten to dot-separated list of IDs

2. Write
2.1 Hugo front matter:
- project: community
- project_version: mailing-list
- description: blank
- sitemap.priority: 0.2
- menu: blank
- toc: false
- aliases: blank

- title: <subject>
- author: <author>
- lastmod: <date> in YYYY-MM-DDTHH:mm:SS-00:00
- id: mailing-list.<thread.greatgrandparent>.<thread.grandparent>.<thread.parent>.<thread.id>

2.2 Content:
<content> converting from <pre> to CRLF blocks.
Find <a> tags and replace with Markdown links
Find <img> tags and replace with Markdown img.
