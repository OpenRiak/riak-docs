Index should be a list. If it \\*was\\* in a list you would have got an error 
{error,&lt;&lt;"{precommit\\_fail,[{unknown\\_field\\_type,&lt;&lt;\\"lives\\"&gt;&gt;}]}"&gt;&gt;} on put. 

&gt; Obj2 = riakc\\_obj:update\\_metadata(Obj, Meta),
