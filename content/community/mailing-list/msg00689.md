&gt; You want to see how to create a client, then create a bucket, then create

&gt; 
&gt; from riak import \\*
&gt;

&gt; bucket = client.bucket("foo")
&gt; obj = bucket.new("some\\_key", {"key":"value"})
&gt; obj.store
&gt; 
&gt; Then, if you want to make sure it go there, do something like:
&gt; 
&gt; curl http://localhost:8098/riak/foo/some\\_key | python -m json.tool
&gt; 
&gt; and you'll get some JSON back.
