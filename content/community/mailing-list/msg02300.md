Well, when it comes down to it, it's just a method. There's a lot less magic 
in Ripple than you'd think. Try this:

def key
 @key ||= UUID.new # Replace with whatever you want, as long as it's a String
end

&gt;&gt;&gt;&gt; terminology for the same concept (and is analogous to "id" in 
&gt;&gt;&gt;&gt; ActiveRecord). If you want to use a non-String value for the key, create

&gt;&gt;&gt;&gt; a defined property act as the key.

&gt;&gt;&gt;&gt; key\\_on :some\\_num
