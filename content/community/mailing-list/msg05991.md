Ben,

Since we didn't build packages for Ubuntu 11.10 32bit, and instead just did
64bit packages, I assume you are using the 32bit .deb from
http://basho.com/resources/downloads/ In that case, that .deb was built
using ubuntu 10.04 and you are probably running into a fairly common issue
with a mismatch in libssl libraries. Ubuntu 11.10 was the first version
that shipped without libssl version 0.9.8 which is needed by the erlang vm
that ships with Riak. Your fix should be as easy as:

sudo apt-get install libssl0.9.8


Install that and try starting Riak again. If you are still having
troubles, let us know.

Thanks,
Jared
