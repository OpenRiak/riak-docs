&gt; In this case you are running a query against Search.  This will only return
&gt; results if you a) have search enabled [2], b) you install the search hook on
&gt; the 'users' bucket [3], and c) you have the schema setup correctly [4].  I
&gt; realize this isn't entirely straightforward and am willing to help if you
&gt; decide you want to use the search functionality.
