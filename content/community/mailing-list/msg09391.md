&gt;&gt;&gt;&gt;&gt; Those gems are available via rubygems, but since they are pre-release 
&gt;&gt;&gt;&gt;&gt; gems they won't show up as the preferred version when navigating to the 
&gt;&gt;&gt;&gt;&gt; gem's page [1, 3]. If you scroll below, and follow the version links [2, 
&gt;&gt;&gt;&gt;&gt; 4] you'll see that those versions are available if you explicitly select 
&gt;&gt;&gt;&gt;&gt; them in your gemfile using the following:
&gt;&gt;&gt;&gt;&gt; 
&gt;&gt;&gt;&gt;&gt; gem 'gemname', 'version'
&gt;&gt;&gt;&gt;&gt; 
&gt;&gt;&gt;&gt;&gt; or by using the following command:
&gt;&gt;&gt;&gt;&gt; 
&gt;&gt;&gt;&gt;&gt; gem install 'gem name' --pre command.
&gt;&gt;&gt;&gt;&gt; 
&gt;&gt;&gt;&gt;&gt; The bundler/rubygems API won't negotiate a pre-release gem if you don't 
&gt;&gt;&gt;&gt;&gt; explicitly specify it in your gemfile.
&gt;&gt;&gt;&gt;&gt; 
&gt;&gt;&gt;&gt;&gt; [1] https://rubygems.org/gems/riak-sessions
&gt;&gt;&gt;&gt;&gt; [2] https://rubygems.org/gems/riak-sessions/versions/1.0.0.beta
&gt;&gt;&gt;&gt;&gt; [3] https://rubygems.org/gems/ripple
&gt;&gt;&gt;&gt;&gt; [4] https://rubygems.org/gems/ripple/versions/1.0.0.beta2
