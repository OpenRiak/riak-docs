&gt;&gt; The Accept header indicates what content-type your client is able to
&gt;&gt; handle. When you specify "application/json" in the Accept header you are
&gt;&gt; telling Riak to return the object if it's content-type is "application/json"
&gt;&gt; otherwise return 406.
